# React useContext Demo
## Create an app by Vite
```
$ yarn create vite react-useContext-demo --template react -y
```

## Deploy to GitHub Pages
* Commands
```
$ yarn build
$ git add -f dist
$ git commit -m "Add dist folder"
$ git subtree push --prefix dist/ origin gh-pages
```
